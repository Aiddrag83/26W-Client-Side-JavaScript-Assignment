# Energy Bowl Builder

A web-based front-end project that allows customers to build and order a customizable energy bowl using HTML, CSS, and object-oriented JavaScript.

## Features

- Structured bowl customization form
- `EnergyBowl` JavaScript class
- Dynamic order receipt on page
- Total price calculation
- Nutrition snapshot
- Local storage for past orders
- Responsive, clean design

## Files

- `index.html` - page structure and form
- `styles.css` - styling and layout
- `script.js` - OOP logic and interactivity

## How to Run

1. run the files
2. run `index.html` in using live server
3. Fill out the form and click **Place Order**
